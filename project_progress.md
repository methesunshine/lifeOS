## 📖 Project Overview
LifeOS is an intelligent Personal Operating System designed to track, analyze, and optimize personal growth across 9 core areas. It transforms daily metrics into actionable insights using an "Atomic" design system and a centralized "Life Score" engine.

### 🧩 Core Modules
1. **Mental & Emotional Health:** Mood, stress, focus tracking.
2. **Physical Health:** Sleep, water, steps, workouts.
3. **Finance:** Credit/Debit transaction logging, savings rate, cash flow.
4. **Skills:** Skill progression and hour logging.
5. **Goals:** Smart goals with subtasks and progress tracking.
6. **Habits:** Daily completion and streak management.
7. **Relationships:** Interaction logs and satisfaction ratings.
8. **Productivity:** Task completion and focus hour analysis.
9. **Reflection:** Daily wins, lessons, and monthly reviews.

## 🛠 Technical Stack
- **Framework:** Next.js 16.1.6 (App Router)
- **Frontend library:** React 19.2.3
- **Styling:** Vanilla CSS + TailwindCSS 4 (Atomic design philosophy)
- **Database:** PostgreSQL via Supabase

---

## 🗄 Database Architecture (Supabase)
The system uses a highly relational PostgreSQL schema with RLS enforced on every table.

- **Primary Tables:** `profiles`, `life_scores`, `mental_health`, `physical_health`, `habits`, `habit_logs`, `finance`, `skills`, `skill_logs`, `goals`, `subtasks`, `relationships`, `productivity`, `reflections`, `reminders`.
- **Schema Location:** `schema.sql` (Root directory)
- **Data Isolation:** All queries filter by `user_id = auth.uid()`.
